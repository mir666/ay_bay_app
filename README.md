**AyBay App** is a mobile application that helps users track their monthly income, expenses, and savings. Users can log in to their account, add transactions for different months, select a month from the dropdown, and download PDF reports.

This app is developed using **Flutter & GetX** and integrated with **Firebase Firestore** for database management.

  Works seamlessly on all screen sizes.
